# Simple-Ai-Agents

Ai-agentsを勉強するため、学習メモを記録する

## 開発環境作成

### Linuxの場合、以下のコマンドで仮想環境を作成してください

'''Bash
python3 -m venv ai-agent-env
source ai-agent-env/bin/activate
pip install requests openai tavily-python pyyaml
'''

### Windowsの場合、以下のコマンドで仮想環境を作成してください

'''Powershell
python3 -m venv ai-agent-env
.\ai-agent-env\Scripts\Activate.ps1
pip install requests openai tavily-python pyyaml
'''
